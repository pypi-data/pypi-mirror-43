## 自然语言常见特征轮 
- 简述：
1. 核心词特征：特征主要由词袋模型，TF-IDF特征为代表。分词算法确定特征粒度。
2. 语序特征：模板方法特征抽取，word2vec含有基于语料顺序的特征。rnn 会学习到部份语序特征。ngram会学习到字成词特征。
3. 意图特征：意图识别，通过监督方法迁移入特征。
4. 主题特征：类似意图特征
5. 其它：字型特征，拼音特征，五笔五角号码特征，字符串特征，知识图谱特征。聚类生成特征。
